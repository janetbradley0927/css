## Code
### Block code
```html
<h1 class="block text:center font:48 tracking:.2em">
    Hello, world!
</h1>
```

### Inline code
- js -> `js const foo = 'bar'`
- ts -> `ts const options: Options = {}`
- css -> `css body { background: red; }`
- mcss -> `mcss text:center`

### Mark added and removed lines
```ts
console.log('hewwo') // [!code --]
console.log('hello') // [!code ++]
console.log('goodbye')
```
