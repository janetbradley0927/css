## Code
### Block code
```html
<h1 className="block text:center font:48 tracking:.2em">
    Hello, world!
</h1>
```

### Inline code
- js -> `js const foo = 'bar'`
- ts -> `ts const options: Options = {}`
- css -> `css body { background: red; }`
- mcss -> `mcss text:center`

### Code lines added and removed
```ts
console.log('hewwo') // [!code --]
console.log('hello') // [!code ++]
console.log('goodbye')
```

### Code line highlight
```ts
console.log('hewwo')
console.log('hello') // [!code highlight]
console.log('goodbye')
```

### Code line focus
```ts
console.log('hewwo')
console.log('hello') // [!code focus]
console.log('goodbye')
```

### Code mark words
```ts
// @MARK bar
const foo = 'bar'
```
