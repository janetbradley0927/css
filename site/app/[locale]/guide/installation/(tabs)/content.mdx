The easiest way to use Master CSS from scratch is to initialize the [runtime engine](/guide/rendering-modes/runtime-rendering) directly in the application's entry file.

## Installation [sr-only]
<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS
            Install Master CSS Runtime into your project via package managers.
        </StepL>
        <StepR>
            ```bash action=install
            @master/css-runtime@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Initialize configuration file
            Run the command to create a configuration file [master.css.js](/guide/configuration).
        </StepL>
        <StepR>
            ```bash action=create
            @master/css@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up CSS runtime engine
            Import and initialize the runtime engine in the entry file, such as `main.js`:
        </StepL>
        <StepR>
            ```js name=main.js
            import { initCSSRuntime } from '@master/css-runtime'
            import config from './master.css'

            initCSSRuntime(config)
            ```
        </StepR>
    </Step>
    <Step>
        ### <StepNum />Start using Master CSS
        Now style your first element using Master CSS syntax!
        ```html name=index.html
        <!DOCTYPE html>
        <!-- @MARK hidden -->
        <html lang="en" hidden>
        <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1">
            <script src="/main.js"></script> <!-- [!code highlight] -->
        </head>
        <body>
            <!-- @MARK font:40 font:heavy italic m:12x text:center -->
            <h1 class="font:40 font:heavy italic m:12x text:center">Hello World</h1>
        </body>
        </html>
        ```
        Open your browser to watch the changes.
        <HelloWorld />
    </Step>
</StepSection>
