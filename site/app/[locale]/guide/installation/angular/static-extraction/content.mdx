## Quick start
<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Clone the example
            Copy-paste the commands to quickly start using the https://angular.se.rc.css.master.co example.

            You can skip all [installation](#installation) steps.
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm create @master/css@rc my-app --example angular-with-static-extraction
            cd my-app
            npm run dev
            ```
        </StepR>
        <StepEnd/>
    </Step>
</StepSection>

## Installation

<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don&#39;t have a Angular project, create one first. It&#39;s recommended to refer [Angular CLI](https://angular.io/cli).
        </StepL>
        <StepR>
            <CodeTabs>
                {[
                    {
                        name: 'Terminal',
                        lang: 'bash',
                        code: `
                            ng new my-app
                            cd my-app
                        `
                    }
                ]}
            </CodeTabs>
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Create a configuration file
            Run the command to create a [master.css.js](/guide/configuration) file.
        </StepL>
        <StepR>
            ```bash action=create
            @master/css@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS and others
            Install the Master CSS and Angular custom webpack plugin into your project via package managers.
        </StepL>
        <StepR>
            ```bash action=install
            @master/css.webpack@rc @angular-builders/custom-webpack
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up CSS extractor
            - Create a `webpack.config.js` file
            - Add a `MasterCSSExtractorPlugin` to the plugins
        </StepL>
        <StepR>
            ```js name=webpack.config.js
            const { MasterCSSExtractorPlugin } = require('@master/css.webpack') // [!code ++]

            module.exports = (config) => { // [!code ++]
                config.plugins.push( // [!code ++]
                    new MasterCSSExtractorPlugin({ // [!code ++]
                        sources: ['src/index.html'] // [!code ++]
                    }) // [!code ++]
                ) // [!code ++]
                return config // [!code ++]
            } // [!code ++]
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Configure Angular
            Use `@angular-builders/custom-webpack` instead of the default extractor for custom webpack configuration.
        </StepL>
        <StepR>
            <CodeTabs>
                {[
                    {
                        name: 'angular.json',
                        lang: 'js',
                        code: `
                                {
                                    …,
                                    "projects": {
                                        "my-app": {
                                            …,
                                            "architect": {
                                                "build": {
                                                     "extractor": "@angular-devkit/build-angular:browser", // [!code --]
                                                     "extractor": "@angular-builders/custom-webpack:browser", // [!code ++]
                                                     "options": {
                                                         "customWebpackConfig": { // [!code ++]
                                                             "path": "./webpack.config.js" // [!code ++]
                                                         }, // [!code ++]
                                                        …
                                                    },
                                                    …
                                                },
                                                "serve": {
                                                     "extractor": "@angular-devkit/build-angular:dev-server", // [!code --]
                                                     "extractor": "@angular-builders/custom-webpack:dev-server", // [!code ++]
                                                     "options": { // [!code ++]
                                                         "browserTarget": "my-app:build" // [!code ++]
                                                     }, // [!code ++]
                                                    …
                                                },
                                                …
                                            }
                                        },
                                        …
                                    }
                                }
                        `
                    }
                ]}
            </CodeTabs>
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Import virtual CSS module
            Import the `master.css` virtual CSS module into the global CSS file `src/styles.css`.

            (!) Don&#39;t import it in the <code>src/main.ts</code>, you&#39;ll get some errors about webpack loaders.
        </StepL>
        <StepR>
            ```css name=src/styles.css
            /* You can add global styles to this file, and also import other style files */
            @import 'virtual:master.css'; /* [!code ++] */
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Launch server
            Run the development server.
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm run start
            ```
        </StepR>
    </Step>
    <Step>
        ### <StepNum />Start using Master CSS
        Now style your first element using Master CSS syntax!
        ```html name=src/app/app.component.html
        <h1 class="italic m:12x fg:strong font:40 font:heavy">Hello World</h1>
        ```
        Open your browser to watch the changes.
        <HelloWorld url="localhost:3000" />
    </Step>
</StepSection>
