<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don't have a Laravel project, create one first. It's recommended to refer to [Installation - Laravel](https://laravel.com/docs/installation)
        </StepL>
        <StepR>
            ```bash name=Terminal
            laravel new my-app
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS
            Install `@master/css.vite` plugin into your project.
        </StepL>
        <StepR>
            ```bash action=install
            @master/css.vite@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Create a configuration file
            Run the command to create a [master.css.js](/guide/configuration) file.
        </StepL>
        <StepR>
            ```bash action=create
            @master/css@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up Master CSS
            Add a `masterCSS` vite plugin in `vite.config.js`.

            - Set `options.mode` to `extract`
            - Add php files to `options.sources`
        </StepL>
        <StepR>
            ```js name=vite.config.js
            import { defineConfig } from 'vite'
            import masterCSS from '@master/css.vite' // [!code highlight]

            export default defineConfig({
                plugins: [
                    masterCSS({ // [!code highlight]
                        mode: 'extract', // [!code highlight]
                        sources: ['resources/views/**/*.php'] // [!code highlight]
                    }) // [!code highlight]
                ]
            })
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Launch server
            Run Laravel and Vite development servers.
        </StepL>
        <StepR>
            ```bash name=Terminal
            composer run dev
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Hello world!
            Now style your first element using Master CSS syntax!
        </StepL>
        <StepR>
            ```html name=resources/views/app.blade.php
            …
            <body>
                <h1 class="font:40 font:heavy italic m:12x text:center">
                    Hello World
                </h1>
            </body>
            …
            ```
        </StepR>
    </Step>
    <HelloWorld url="127.0.0.1:8000" />
</StepSection>
