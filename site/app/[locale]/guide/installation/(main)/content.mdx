Quickly create a local-launched project by specifying a [rendering mode](/guide/rendering-modes) via the Master CSS Vite plugin.

## Installation [sr-only]
<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a Vite project
            If you don't have a Vite project, create one first. It's recommended to refer to [Getting Started - Vite](https://vite.dev/guide/)
        </StepL>
        <StepR>
            ```bash action=create
            vite@latest my-project
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS
            Install Master CSS vite plugin into your project via package managers.
        </StepL>
        <StepR>
            ```bash action=install
            @master/css.vite@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up Master CSS
            Import the `masterCSS` plugin in the `vite.config.js`.

            - By default, the `options.mode` is set to `runtime`.
            - To create a [master.css.js](/guide/configuration) file, run `npm create @master/css@rc`
        </StepL>
        <StepR>
            ```js name=vite.config.js
            import { defineConfig } from 'vite'
            import masterCSS from '@master/css.vite' // [!code highlight]

            export default defineConfig({
                plugins: [
                    masterCSS() // [!code highlight]
                ]
            })
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Start dev server
            Run the command to start the Vite dev server.
        </StepL>
        <StepR>
            ```bash name=Terminal&action=run
            dev
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Start using Master CSS
            Now style your first element using Master CSS syntax!
        </StepL>
        <StepR>
            ```html name=index.html
            <h1 class="font:40 font:heavy italic">Hello World</h1>
            ```
        </StepR>
    </Step>
    <HelloWorld url="localhost:5173" />
</StepSection>
