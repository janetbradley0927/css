<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don't have a Nuxt.js project, create one first. It's recommended to refer to [Get started with Nuxt.js](https://nuxt.com/docs/getting-started/installation).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npx nuxi@latest init my-app
            cd my-app
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS
            Install `@master/css.nuxt` plugin into your project.
        </StepL>
        <StepR>
            ```bash action=install
            @master/css.nuxt@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Create a configuration file
            Run the command to create a [master.css.js](/guide/configuration) file.
        </StepL>
        <StepR>
            ```bash action=create
            @master/css@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up Master CSS
            Add a `@master/css.nuxt` module in `nuxt.config.ts`.

            - By default, `options.mode` is set to `progressive`.
        </StepL>
        <StepR>
            ```js name=nuxt.config.ts
            export default defineNuxtConfig({
                modules: [
                    '@master/css.nuxt' // [!code highlight]
                ]
            })
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Launch server
            Run `npm run dev` to start your Nuxt.js development server
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm run dev
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Start using Master CSS
            Now style your first element using Master CSS syntax!
        </StepL>
        <StepR>
            ```vue name=app.vue
            <template>
                <h1 class="italic m:12x fg:strong font:40 font:heavy">
                    Hello World
                </h1>
            </template>
            ```
        </StepR>
    </Step>
    <HelloWorld url="localhost:3000" />
</StepSection>
