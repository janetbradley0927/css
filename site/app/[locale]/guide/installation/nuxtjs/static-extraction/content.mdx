    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don't have a Nuxt.js project, create one first. It's recommended to refer to [Get started with Nuxt.js](https://nuxt.com/docs/getting-started/installation).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npx nuxi@latest init my-project
            cd my-project
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Initialize configuration file
            Run the command to create a configuration file [master.css.ts](/guide/configuration).
        </StepL>
        <StepR>
            ```bash action=create
            @master/css@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS
            Install the Master CSS builder into your project via package managers.
        </StepL>
        <StepR>
            ```bash action=install
            @master/css.vite@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up CSS extractor
            Add a `@master/css.nuxt` to the nuxt modules

            - Set `options.mode` to `extract`
        </StepL>
        <StepR>
            ```ts name=nuxt.config.ts
            export default defineNuxtConfig({
                modules: [
                    ['@master/css.nuxt', { mode: 'extract' }] // [!code ++]
                ]
            })
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Launch server
            Run `npm run dev` to start your Nuxt.js development server
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm run dev
            ```
        </StepR>
    </Step>
    <Step>
        ### <StepNum />Start using Master CSS
        Now style your first element using Master CSS syntax!
        ```html name=app.vue

        <template>

        </template>
        ```
        Open your browser to watch the changes.
        <HelloWorld url="localhost:3000" />
