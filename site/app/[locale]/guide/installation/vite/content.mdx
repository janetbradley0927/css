<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don't have a Vite project, create one first. It's recommended to refer to [Getting Started - Vite](https://vitejs.dev/guide/)
        </StepL>
        <StepR>
            ```bash action=create
            vite@latest my-project
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS
            Install the Master CSS vite plugin into your project via package managers.
        </StepL>
        <StepR>
            ```bash action=install
            @master/css.vite@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Initialize configuration file
            Run the command to create a configuration file [master.css.ts](/guide/configuration).
        </StepL>
        <StepR>
            ```bash action=create
            @master/css@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up Master CSS
            Import the `masterCSS` plugin in the `vite.config.js`.

            - By default, the `options.mode` is set to `runtime`.
            - To create a [master.css.js](/guide/configuration) file, run `npm create @master/css@rc`
        </StepL>
        <StepR>
            ```js name=vite.config.js
            import { defineConfig } from 'vite'
            import masterCSS from '@master/css.vite' // [!code highlight]

            export default defineConfig({
                plugins: [
                    masterCSS() // [!code highlight]
                ]
            })
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Start dev server
            Run the command to start the Vite dev server.
        </StepL>
        <StepR>
            ```bash name=Terminal&action=run
            dev
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Hello world!
            Now style your first element using Master CSS syntax!
        </StepL>
        <StepR>
            ```html name=index.html
            <!DOCTYPE html>
            <html lang="en">
                …
                <body>
                    <h1 class="font:40 font:heavy italic m:12x text:center">Hello World</h1>
                    …
                </body>
            </html>
            ```
        </StepR>
    </Step>
    <HelloWorld url="localhost:5173" />
</StepSection>
