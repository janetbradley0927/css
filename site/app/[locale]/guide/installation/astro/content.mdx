<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don't have a Astro project, create one first. It's recommended to refer to [Getting Started - Astro](https://docs.astro.build/en/getting-started).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm create astro@latest my-app
            cd my-app
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS
            Install `@master/css.astro` plugin into your project.
        </StepL>
        <StepR>
            ```bash action=install
            @master/css.astro@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Create a configuration file
            Run the command to create a [master.css.js](/guide/configuration) file.
        </StepL>
        <StepR>
            ```bash action=create
            @master/css@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up Master CSS
            Add a `masterCSS` astro integration in `astro.config.js`.

            - By default, `options.mode` is set to `runtime`.
        </StepL>
        <StepR>
            ```js name=astro.config.js
            import { defineConfig } from 'astro/config'
            import masterCSS from '@master/css.astro' // [!code highlight]
            import sitemap from '@astrojs/sitemap'

            // https://astro.build/config
            export default defineConfig({
                compressHTML: true,
                trailingSlash: 'never',
                integrations: [
                    sitemap(),
                    masterCSS() // [!code highlight]
                ]
            })
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Start dev server
            Run the command to start the Astro dev server.
        </StepL>
        <StepR>
            ```bash name=Terminal&action=run
            dev
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Start using Master CSS
            Now style your first element using Master CSS syntax!
        </StepL>
        <StepR>
            ```astro name=src/pages/index.astro
            ---
            import Layout from '../layouts/Layout.astro';
            ---
            <Layout title="Welcome to Astro.">
                <main>
                    <h1 class="italic m:12x fg:strong font:40 font:heavy">
                        Hello World
                    </h1>
                </main>
            </Layout>
            ```
        </StepR>
    </Step>
    <HelloWorld url="localhost:3000" />
</StepSection>
