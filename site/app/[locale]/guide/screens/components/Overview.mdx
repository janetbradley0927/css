For example, a screen size token can be used in the following ways:
```html
<!-- @MARK @md -->
<div class="font:32@md">…</div>
<!-- @MARK screen-md -->
<div class="max-w:screen-md">…</div>
<!-- @MARK @container(md) -->
<div class="hidden@container(md)">…</div>
