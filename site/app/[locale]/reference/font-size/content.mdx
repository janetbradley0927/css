### With line height
Set the line height, use the [`leading:<size>`](/reference/line-height) syntax along with the font size.
```html
<!-- @MARK font:md leading:1.2 -->
<div class="font:md leading:1.2">Sphinx of black quartz, judge my vow.</div>
```
You can also use the [`text:<size>`](/reference/text-size) syntax to set the line height and font size together.
