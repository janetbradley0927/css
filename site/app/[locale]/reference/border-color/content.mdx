## Customization
## Use custom border colors
Add a color to the line variable namespace.
```js name=master.css.js
export default {
    modes: {
        light: {
            color: {
                line: {
                    light: '$slate-60/.2' /* [!code highlight] */
                }
            }
        },
        dark: {
            color: {
                line: {
                    light: '$gray-30/.2' /* [!code highlight] */
                }
            }
        }
    }
}
```
Apply the defined `line-*` variables using the inherited rules <FrameHeirs />.
```html
<div class="outline:neutral"></div> <!-- [!code highlight] -->
<div class="outline:line-neutral">equals</div>
<div class="outline:$line-neutral">equals</div>
```
