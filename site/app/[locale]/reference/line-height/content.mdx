## Examples
### Basic usage
Use the predefined line height tokens like `leading:xs`, `leading:sm`, and `leading:lg` to set the line height of an element.
<Demo $py={0}>
    <DemoPanel className="flex flex-col gap:8x w:full">
        <div className="w:full">
            <DemoLabel>leading:xs</DemoLabel>
            <DemoP className="font:md leading:xs">Design is not just about aesthetics—it’s about communication. Proper font sizes and spacing help present information clearly and improve usability.</DemoP>
        </div>
        <div className="w:full">
            <DemoLabel>leading:sm</DemoLabel>
            <DemoP className="font:md leading:sm">Design is not just about aesthetics—it’s about communication. Proper font sizes and spacing help present information clearly and improve usability.</DemoP>
        </div>
        <div className="w:full">
            <DemoLabel>leading:md</DemoLabel>
            <DemoP className="font:md leading:md">Design is not just about aesthetics—it’s about communication. Proper font sizes and spacing help present information clearly and improve usability.</DemoP>
        </div>
        <div className="w:full">
            <DemoLabel>leading:lg</DemoLabel>
            <DemoP className="font:md leading:lg">Design is not just about aesthetics—it’s about communication. Proper font sizes and spacing help present information clearly and improve usability.</DemoP>
        </div>
        <div className="w:full">
            <DemoLabel>leading:xl</DemoLabel>
            <DemoP className="font:md leading:xl">Design is not just about aesthetics—it’s about communication. Proper font sizes and spacing help present information clearly and improve usability.</DemoP>
        </div>
    </DemoPanel>
</Demo>
```html
<!-- @MARK leading:xs -->
<div class="leading:xs">Design is not just about aesthetics—it’s...</div>
<!-- @MARK leading:sm -->
<div class="leading:sm">Design is not just about aesthetics—it’s...</div>
<!-- @MARK leading:md -->
<div class="leading:md">Design is not just about aesthetics—it’s...</div>
<!-- @MARK leading:lg -->
<div class="leading:lg">Design is not just about aesthetics—it’s...</div>
<!-- @MARK leading:xl -->
<div class="leading:xl">Design is not just about aesthetics—it’s...</div>
```
You can also use [`text:<size>`](/reference/text-size) to set a text size with a golden ratio line height.

---

## Conditionally apply
Apply styles based on different states using [selectors](/guide/state-selectors) and [conditional queries](/guide/conditional-queries).
```html
<div class="leading:1:hover leading:1@sm leading:1@dark leading:1@print">…</div>
```
