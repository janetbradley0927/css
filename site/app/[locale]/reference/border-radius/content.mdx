## Customization
### Use custom border radius
Customize your border radius tokens by defining [variables](/guide/variables).
```js name=master.css.js
export default {
    variables: {
        'border-radius': {
            xs: 2, /* [!code ++] */
            sm: 4, /* [!code ++] */
        }
    }
}
```
Apply the defined `border-radius` variables.
```html
<!-- @MARK r:sm -->
<div class="r:sm">…</div>
```
